# Donkey Kong Game - C++

## **Overview**

**This is a Donkey Kong-inspired game developed in C++. The game features platforming mechanics, enemy interactions, and a classic arcade-style experience.**

## **Features**

- **Classic platformer gameplay**
- **Multiple levels with increasing difficulty**
- **Enemy AI and obstacles**
- **Score tracking and high-score system**
- **Smooth animations and responsive controls**

## **Installation**

### **Prerequisites**

- **C++ compiler (GCC, Clang, or MSVC)**
- **SFML library (if using for graphics and input handling)**

### **Steps**

1. **Clone this repository:**
   
   git clone https://github.com/Slizerix/donkey-kong-cpp.git
   cd donkey-kong-cpp
  
2. **Install dependencies:**
   - **If using SFML, download and install it from **[**SFML's official site**](https://www.sfml-dev.org/)
3. **Compile the game:**
  
   g++ -o donkey_kong main.cpp Game.cpp Player.cpp Enemy.cpp -lsfml-graphics -lsfml-window -lsfml-system
   ```
4. **Run the game:**
 
   ./donkey_kong
